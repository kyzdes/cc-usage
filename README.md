# CC Usage Viewer — distribution mirror

This repository hosts **release assets and auto-update feeds** for [CC Usage Viewer](https://cc-usage-widget.com) — a native menu-bar / tray app that shows your Claude subscription usage.

## Download

- **Website:** https://cc-usage-widget.com
- **Releases:** see the [Releases](https://github.com/kyzdes/cc-usage/releases) tab for Windows installers (`CCUsageViewer-Setup.exe`).
- **macOS builds** live in [`kyzdes/claude-usage`](https://github.com/kyzdes/claude-usage/releases).

## Auto-update feeds

- `appcast-windows.xml` — AutoUpdater.NET feed for the Windows app.
- `appcast.xml` — mirror of the Sparkle feed for the macOS app (live feed: [`kyzdes/claude-usage/appcast.xml`](https://raw.githubusercontent.com/kyzdes/claude-usage/main/appcast.xml)).
